# Articles Overview

Welcome to the **Perplexity Sonar API Articles** collection! These are in-depth guides and tutorials that cover advanced implementation patterns and integrations with other tools and frameworks.

## 📚 Available Articles

### 🧠 [Memory Management](memory-management/)

**Purpose**: Advanced solutions for preserving conversational memory in LLM applications  
**Focus**: Context window management and long-term memory persistence

**Topics Covered**:
- **Chat Summary Memory Buffer** - Summarize older messages to fit within token limits while retaining key context
- **Persistent Memory with LanceDB** - Enable long-term memory persistence across sessions using vector embeddings

**Key Benefits**:
- 43% reduction in token usage through summarization
- 92% context retention across sessions
- Production-ready scalable architectures

**Quick Start**:
```bash
cd memory-management/chat-summary-memory-buffer/
python chat_summary_memory.py
```
