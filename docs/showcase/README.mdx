## 🤝 Submit Your Project

Have a project built with the Sonar API? We'd love to feature it in our showcase!

### How to Submit

1. **Build your project** in a separate public repository
2. **Fork this cookbook** repository
3. **Create a new MDX file** under `/docs/showcase/your-project-name.mdx`
4. **Follow the template** structure (title, description, features, installation, etc.)
5. **Submit a pull request** with your contribution

### Requirements

- Clear documentation with setup instructions
- Working demo or screenshots
- Proper attribution and licensing
- Novel use case that adds value to the community

See our [Contributing Guidelines](https://github.com/perplexityai/api-cookbook/blob/main/CONTRIBUTING.md#for-showcase-projects) for detailed submission instructions.
